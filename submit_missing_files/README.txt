Prepare config files for treatment and submition

- Check which datasets are incomplete
- Run /home/hep/jtafoyav/production_2023/tools/get_file_report/get_report.sh on the relevant files to get the list of processed files
- Copy the relevant crab_*.py config files to a new directory e.g. /home/hep/jtafoyav/production_2023/CMSSW_13_0_14/src/2023_DIGIRAW-ext/_resubmit/
- Go to the _resubmit directory containing the .py files and run /home/hep/jtafoyav/production_2023/tools/submit_missing_files/adjust_config.py on all of them
- Now you can do crab submit on all of them


To copy the relevant files, you can do:
while read p; do
  DATASET=$p
  cp crab_submit_${DATASET}.py _resubmit/
done <list_to_resubmit.txt

where "list_to_resubmit.tx" is generated by the get_report.sh script
